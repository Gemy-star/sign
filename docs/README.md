# Logo Files Setup

Please add the following logo files to the `static/images/` directory:

## Required Files:

1. **logo.png** - Main logo for the sidebar (light version)
   - Recommended size: 200x60 pixels
   - Format: PNG with transparent background
   - Used in: Dashboard sidebar

2. **logo-dark.png** - Dark logo for login page
   - Recommended size: 200x60 pixels
   - Format: PNG with transparent background
   - Used in: Login page
   - This logo will be inverted to white color via CSS filter

## Directory Structure:
```
static/
  images/
    logo.png
    logo-dark.png
```

## Quick Create Placeholder:

If you don't have logos yet, you can create simple text-based placeholders or use any image temporarily.

For production, make sure to replace these with your actual brand logos.
